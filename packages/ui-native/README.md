# packages/ui-native

**Atlas React Native component library.**

Stack: React Native · Expo · Atlas token bridge (`tokens/atlas.tokens.ts`)

## Structure
```
components/   — 12 Atlas components (native implementations)
theme/        — ThemeProvider, useTheme hook
tokens/       — atlas.tokens.ts (CSS vars bridged to JS object)
utils/        — shadowHelper, useReducedMotion
```

## Running
```bash
cd packages/ui-native
npx expo start
```
